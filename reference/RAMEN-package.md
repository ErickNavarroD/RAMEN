# RAMEN: RAMEN: Regional Association of Methylome variability with the Exposome and geNome

R package that identifies which genetic (G), environmental (E), additive
(G+E) or interaction (GxE) effect better explains DNA methylation levels
in Variable Methylated Loci using microarray data.
