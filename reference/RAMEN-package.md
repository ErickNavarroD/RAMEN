# RAMEN: RAMEN: Regional Association of Methylome variability with the Exposome and geNome

Using population data, RAMEN identifies which genetic (G), environmental
(E), additive (G+E) or interaction (GxE) model better explains DNA
methylation levels in genome-wide locations with high DNA methylation
variability.
