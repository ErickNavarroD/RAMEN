# Genotype metadata example data set

Genotype chromosome 21 position metadata obtained from a private
genotyping data set.

## Usage

``` r
test_genotype_information
```

## Format

### `test_genotype_information`

A data frame with 8539 rows and 3 columns:

- CHROM:

  Chromosome

- POS:

  Probe genomic position (h19)

- ID:

  SNP ID
